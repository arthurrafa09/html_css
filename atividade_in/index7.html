Perguntas Frequentes

O que é HTML?

HTML é a linguagem padrão para criar páginas web e aplicações web.

O que é CSS?

CSS é uma linguagem usada para descrever a apresentação de um documento escrito em HTML.

O que é JavaScript?

JavaScript é uma linguagem de programação que permite implementar funcionalidades complexas em páginas web.

Qual a diferença entre section e article

Um article é um conteúdo independente e autocontido, enquanto uma section é um agrupamento temático de conteúdo.

Para que serve a tag aside

A tag aside é usada para conteúdo tangencialmente relacionado ao conteúdo principal da página, como uma barra lateral.
